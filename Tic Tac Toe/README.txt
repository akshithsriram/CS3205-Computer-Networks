The files corresponding to the first question are : gameserver.cpp and gameclient.cpp
The file corresponding to the second question is: yapp.cpp

Instructions for compilation & running:
	- To compile all the c++ code files, run the command "make all"
	- To delete previously available executables, run "make clean"
	- To execute the tic tac toe game-server, use
		$ ./gameserver
		$ ./gameclient
	- To test the yapp program, use
		$ ./yapp 8.8.8.8
	  where 8.8.8.8 can be replaced by any valid IPv4 address
